# 💱 Currency Converter

A responsive and user-friendly **Currency Converter** web application built using **HTML, CSS, and JavaScript**. It allows users to convert currencies instantly using real-time exchange rates through an API.

## 📌 Features

-  Convert between multiple international currencies
-  Real-time exchange rate data
-  Instant currency conversion
-  Clean and responsive user interface
-  Fast and lightweight application

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/currency-converter.git
```

2. Navigate to the project folder

```bash
cd currency-converter
```

3. Open `index.html` in your browser.

## 💡 How It Works

1. Enter the amount.
2. Select the source currency.
3. Select the target currency.
4. Click the **Convert** button.
5. The application fetches the latest exchange rate and displays the converted amount instantly.
